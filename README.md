# Custom Cuisine Catering
Website for Rob Wilson's catering service: http://www.customcuisine.com/.

## Usage

### Developer Usage
Fire it up:

```bash
rvm use ruby 2.5.1 && \
  bundle install && \
  bundle exec jekyll serve --force_polling --livereload
```

Site should ultimately be at http://127.0.0.1:4000/.
